1. Understanding Fluid Helium 

The main goal of this library is to help you model your own cryogenic system on Matlab/Simulink, using a multi-physical approach. This mean that all the blocks in this library are really cloth to the real component.

The best way to understand the principle of multi-physical modelling and how to use this library is to see some examples. The links below explain all this, starting with basics explanations and finishing with the modelling of a complete cryogenic system.
