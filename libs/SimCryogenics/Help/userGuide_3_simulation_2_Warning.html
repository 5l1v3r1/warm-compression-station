3.2. Simulation warning 

Using this library, you can have warnings printing in the console during the simulation. There is two different types of warnings :

  • Control input warning
    • In this case, you should have a message like this :

    The speed velocity of the control input is lower than ControlInputMin

    This is the message that the compressor will print if the actual control input is out of the range defined in the Constraints panel of the compressor
    (see Parametrize your component section for more information). If you want to avoid this, you can reset the maximum and minimum limitation of the compressor, or you can set the actual control input inside of the range allowed for the component.

  • Physical data value warning
    • If you have a message like this :

    The output pressure 1 is too high (>P_max)

    it's mean that the actual value of the physical data at the input or output of your component is not in the allowed range define in the fluid properties block
    In the warning you will have the following information :
    • The localisation of the block
    • The localisation of the port
    • The kind of physical data (M,P or T)
    • The type of problem (too high or too low)
