3.3. Simulation error 

There is one case of error that you can have : wrong setting for a block.
If you don't respect the instruction at the head of each block, you can set a wrong value in the block parameter.
In this case you should have a message like this :

Constant of the compressor must be greater than zero

Here the simulation will stop. So you have to correct the wrong parameter in the block and restart the simulation.
All the other error that you can have are not depending on this library.

For more information about errors and warnings please have a look at the Troubleshooting section
