4.2. Initialisation problems 

An other current mistake came from the initialization. Sometime Matlab will display an error message, explaining that the solver can't solve the initial condition.
Generally, in this case the problem cameS from the difference between the initial conditions parameter and the real initial conditions.

For example, if you set the initial pressure of a pipe at 16 bar,
then you have to be sure that the input of the pipe is connected to an element that provides 16 bar of pressure(it could be a bit more or less but not that much).
If it's not the case, they will be a huge derivative between the Initial conditions parameter and the real initial conditions, so the solver can't deal with it.

So if you use components that have initial conditions, be sure that they are correctly set.
