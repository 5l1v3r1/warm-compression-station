Fluid helium 1.0

1. What is Fluid helium toolbox ? 

The Fluid helium toolbox is a library that allowed you to create your own model of cryogenic systems. This library is based on the Matlab toolbox Simscape which use the multi-physical modelling approach. The main functionalities are :

  • Functional elementary block such as compressor, turbine and many more
  • Customization of the fluid properties
  • Possibility to implement your own block
