2.3. Parametrize your component 

a. Basic parameters

Each blocks of the elements section, can be parametrized. You just have to double click on a block to open the graphical panel that allows you to parametrized your component.
Here is the one of the pipe :



In the first panel (Parameters) you can set the parameters of the component (here the volume).

b. Initial conditions

Some component have initial conditions, if you want to set this initial conditions you just have to go in the Initial conditions panel.
Be careful, if you set an initial condition, you have to be sure that at the beginning of the simulation, the actual input or output of the component (depend on the component), is equal to the initial condition.




c. Constraints

Finally, each blocks have a Constraints panel. In this panel you can set the constraints imposed to the block.
In the following example, you can see the constraint imposed to the input control of the compressor.



Here, the compressor control must be comprised between 0.001 rad/s and 10 rad/s If the input control is not on this range, a warning message will be display in the console.
