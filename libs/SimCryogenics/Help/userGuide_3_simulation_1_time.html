3.1. Time of the simulation 

Simulation time can vary from one schema to another. For example, you can use ideal or non ideal component.
If you choose to use the ideal one, there will be less equations to solve so the time of the simulation should decrease.
But the main time of the simulation is the building time, it's the time that your compiler takes to compile the model.
If you want to avoid this time, you should use the fast restart option.
With this option you can simulate faster, but you can only change the value of the input of your circuit.
In this case, it's impossible to change the parameters of your blocks.

Sometime, the simulation can be very slow at the beginning or just after an input drop.
It's because of an important derivative. Generally you just have to wait a few time and the simulation go faster.
