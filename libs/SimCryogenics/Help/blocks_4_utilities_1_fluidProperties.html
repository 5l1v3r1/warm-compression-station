Fluid helium 1.0

4.1. Fluid properties 

a. Description

The block controls fluidHelium domain properties for the attached circuit. You must have one of this block connecting to your circuit.
This bloc provides all the parameter of you physical environment to your schema.

b. Instructions

To ensure compilation, you must respect the following points :

  • All the parameters have to be greater than 0
  • Minimum parameters have to be lower than maximum parameters
  • If you use a very large scale (ie Min << Max), the precision of all your system may decline
