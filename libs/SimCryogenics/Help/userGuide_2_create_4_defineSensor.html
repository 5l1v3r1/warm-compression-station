2.4. Define sensors 

a. Use sensors

If you want to get result from your simulation, you should implement sensors on your circuit.
All the sensors are perfect, that mean that they don't influence the circuit at all.
There is four type of sensor :

  • Mass flow sensor
  • Pressure sensor
  • Temperature sensor
  • Quality (phase of the fluid)
    • For more information about this sensors, you should check the sensors section

b. Use explorer

Another way to get data from your simulation is to use the Matlab command sscexplore()
Go to the Model Configuration Parameters panel on your Simulink file. Go to the section named Simscape, and change the log simulation data from None to All
Then you can simulate your system and write this in the Matlab console : sscexplore(simlog)
