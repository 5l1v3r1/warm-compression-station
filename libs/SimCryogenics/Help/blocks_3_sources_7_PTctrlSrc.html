3.7 P/T controlled sources 

a. Description

This block is a perfect pressure and temperature source. It imposes a constant pressure and temperature regardless of the mass flow.
The first input of this component defines the value of the pressure, the unit of the mass flow is [bar].
The second input of this component defines the value of the temperature, the unit of the temperature is [K].
