2.1. Define a source 

a. What is a source ?

A source is an component that provides physical data to the system.
All the sources are ideal in a sense that it's assumed to provide a perfect constant physical data independently of the others physical data.
There is different kind of sources in this library, as there is different physical data.
Here is the list of the different sources :

  • Mass flow
  • Mass flow and temperature
  • Pressure
  • Pressure and temperature

b. What kind of source should I use ?

To answer this question you have to do two choices :

Firstly, what kind of physical data do you need ? If at the input of you circuit you have a component that requires pressure and temperature, you should pick the pressure and temperature source.
And it's the same for the output, if you have an open output port in your circuit, you should connect it to the appropriate source.

Secondly, you have to choose between a controlled source or a parametrised one.
If the value of the source doesn't evolve during the time, you should pick a non controlled source.
But if you want to control the value provided by the source dynamically, with a step for example, you should use a controlled source.
This kind of sources are recognizable by their additional port(s).
the form of the port is a little triangle. You can't directly connect a Simulink control signal to this port. You have to use a Simulink-PS converter..

For more information about sources, please read the sources section..
